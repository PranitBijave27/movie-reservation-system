# Movie Reservation System Backend

Backend service for a scalable movie ticket booking system built with Node.js, Express, and MongoDB.

## Features implemented
- Movie management
- Theater management
- Screen creation with automatic seat generation

## Tech Stack
- Node.js
- Express.js
- MongoDB
- Mongoose

## Project Structure
src/
  models/
  controllers/
  services/
  routes/
  middleware/
  config/

## How to run

Install dependencies:
npm install

Run server:
npm run dev
